# Site assets

Drop client-provided files here to complete open tasks in `WEBSITE_NOTES.md`.

| File | Purpose |
|------|---------|
| `images/logo.png` | Header logo (active) |
| `images/hero.jpg` | Home hero still / poster |
| `models/hero.glb` | Interactive WebGL hero (via `<model-viewer>`) |
| Portfolio images | Reference from `data/projects.json` |

Until `hero.jpg` exists, the home page falls back to a stock image.
